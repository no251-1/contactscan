下面我来系统地、深入地介绍 **PVC（Passive Voltage Contrast）技术**，它是一种在**扫描电子显微镜（SEM）**中常用的失效分析和电性诊断手段。

---

## 🧠 一、PVC 的基本概念

**PVC（Passive Voltage Contrast，静态电压对比）**
是一种基于 **扫描电子显微镜（SEM）二次电子成像** 的技术，通过观察样品表面不同区域的亮暗对比，来间接反映这些区域的**电势差异**。

> 简单理解：
> 在 SEM 下，同一个金属线或者晶体管区域，因为电位高低不同，会表现出不同的灰度。
> 电位高的区域 → 发射二次电子多 → 显得亮
> 电位低或接地的区域 → 发射二次电子少 → 显得暗

因此，**PVC 实际上是一种“非接触式的电势映射”方法**。

---

## 🔬 二、PVC 的工作原理

### 1. 电子束与电势的相互作用

SEM 中的电子束轰击样品时，会在样品表面产生：

* **二次电子（SE）**：能量低，来自样品表面；
* **背散射电子（BSE）**：能量较高，来自样品内部。

其中，PVC 主要依赖 **二次电子信号**。

如果样品表面有不同的电势（V₁, V₂），那么：

* 电位较高的区域（正电势）会**排斥入射电子束**，导致**发射角变大、二次电子更易逸出** → 图像较亮；
* 电位较低或接地的区域会**吸引电子**，使二次电子不易逃逸 → 图像较暗。

这就是“电压对比”成像的基础。

---

### 2. 被动（Passive）的含义

PVC 中的 “Passive” 表示：

> 在测量过程中 **不主动对样品施加外部偏压**。

样品内部的电势分布完全取决于其自身的电路状态：

* 哪些节点浮空；
* 哪些节点接地；
* 哪些节点连接到高电位。

因此，PVC 是一种**静态观察技术**，与 **主动电压对比（AVC, Active Voltage Contrast）** 不同。
后者会在测试时主动对电路加偏压，以观察电位变化。

---

## ⚙️ 三、PVC 的应用场景

PVC 技术在 **集成电路失效分析（FA）** 和 **工艺验证** 中被广泛使用：

| 应用方向               | 说明                                     |
| ------------------ | -------------------------------------- |
| **1. 电路节点开路/短路检测** | 开路节点会浮空，导致局部电势异常，从而表现为亮/暗异常。           |
| **2. 晶体管开关状态判断**   | NMOS/PMOS 的源漏区在导通与关断状态下电位不同，可用 PVC 判别。 |
| **3. 金属互连层连通性验证**  | 连通不良或断线的区域可通过亮度不一致发现。                  |
| **4. 静电损伤（ESD）分析** | ESD 破坏可能改变局部电势分布。                      |
| **5. 制程调试/节点识别**   | 快速定位不同功能节点（VDD、GND、信号线等）。              |

---

## 🧩 四、样品准备要求

PVC 对样品的导电性和接地处理非常敏感。
典型的样品制备包括：

1. **去除钝化层（depassivation）**

   * 通过 FIB 或化学蚀刻暴露金属层；
   * 否则电子束无法与内部节点有效相互作用。

2. **局部接地**

   * 样品的 GND 节点必须可靠接地；
   * 否则整个芯片可能浮电，无法形成对比。

3. **充电控制**

   * 过度充电会造成图像漂移或假对比；
   * 通常使用低束流（几十 pA）和低加速电压（<2 kV）。

## 📸 五、PVC 图像的典型特征

| 区域类型      | 电气状态        | SEM 亮度  | 解释           |
| --------- | ----------- | ------- | ------------ |
| 接地线 (GND) | 低电势         | 暗       | 吸引电子，二次电子难逸出 |
| 电源线 (VDD) | 高电势         | 亮       | 排斥电子，二次电子易逸出 |
| 浮空节点      | 不稳定，亮度中间或闪烁 | 介于亮与暗之间 | 受束流注入电荷影响    |
| 短路节点      | 与邻近区域电位相同   | 亮度一致    | 无对比度差        |

## ⚡ 六、PVC 与其他电性分析技术的比较

| 技术                                                  | 原理           | 优点              | 局限            |
| --------------------------------------------------- | ------------ | --------------- | ------------- |
| **PVC (Passive Voltage Contrast)**                  | SEM 下观察自然电位差 | 非接触、快速、定位准      | 电压定量困难、依赖表面状态 |
| **EBIC (Electron Beam Induced Current)**            | 电子束激发电流信号    | 可测 PN 结特性、电性活性区 | 需要电流通路、复杂布线   |
| **OBIRCH (Optical Beam Induced Resistance Change)** | 激光加热导致电阻变化   | 定位短路/漏电缺陷       | 仅限导电路径        |
| **EMMI (Emission Microscopy)**                      | 观察热/光发射      | 无需制样，快速定位       | 空间分辨率较低       |
| **FIB + PVC**                                       | 局部截面 + 电位对比  | 可精确判断连通性        | 成本高，破坏性       |

## 🧮 七、PVC 的物理模型（简化理解）

在 PVC 成像中，二次电子产额 δ 与样品表面电位 Vₛ 的关系可用经验模型表示：

[
\delta = \delta_0 + k \cdot V_s
]

其中：

* δ₀：中性状态下的二次电子发射率；
* k：比例常数；
* Vₛ：样品表面电势。

成像亮度 ∝ δ，
因此 SEM 图像亮度大致反映了表面电势分布。

---

## 🧠 八、实践注意事项

* 选择 **低束流、低电压（1–3 kV）** 避免充电伪像；
* 使用 **场发射SEM (FE-SEM)** 可获得更清晰电位对比；
* 保证样品接地良好；
* 对 CMOS 电路，需了解工艺结构，正确解释亮暗。
